# Лабораторная работа № 1. Основы работы с виртуальной машиной

**Среда:** VirtualBox или VMware. **Образ:** Ubuntu Server.

**Требование:** при развёртывании и настройке ВМ необходимы фотографии (скриншоты) ключевых этапов.

---

## Задания

1. Развернуть виртуальную машину из образа Ubuntu Server в VirtualBox или VMware.

2. Создать обычного пользователя с именем, совпадающим с фамилией студента (латиницей, в нижнем регистре, например `ivanov`).

3. Настроить SSH-сервер на виртуальной машине, организовать доступ этому пользователю по ключу и отключить парольную аутентификацию. Дальнейшую работу выполнять в рамках SSH-сессии.

4. Изучить основные конфигурации виртуальной машины (CPU, память, диски, сеть).

5. Создать новый раздел на свободном месте диска `/dev/sda`, создать файловую систему ext4 и смонтировать новый раздел в `/disk` на постоянной основе. Обеспечить доступность поддерева файловой системы `/disk` созданному пользователю (по фамилии студента).

---

## Шаблон отчёта

1. **Среда выполнения**
   - Гипервизор (VirtualBox / VMware), версия.
   - Образ: Ubuntu Server (версия, если известна).

2. **Задание 1. Развёртывание ВМ**
   - Фотографии (скриншоты): создание ВМ, установка/первый запуск Ubuntu Server.

3. **Задание 2. Пользователь по фамилии студента**
   - Команды и/или скриншот вывода (например, `id <фамилия>`).

4. **Задание 3. SSH по ключу**
   - Команды настройки; скриншот успешного входа по SSH с хоста.

5. **Задание 4. Конфигурация ВМ**
   - Выводы команд (CPU, RAM, диски, сеть) или скриншоты.

6. **Задание 5. Раздел и монтирование `/disk`**
   - Команды разметки и монтирования; вывод `lsblk`, `mount`, фрагмент `/etc/fstab`; скриншот доступа созданного пользователя к `/disk`.

7. **Выводы**
   - Кратко: что сделано, с какими трудностями столкнулись.

---

## Критерии оценивания

| Задание | Критерий |
|--------|----------|
| 1 | ВМ развёрнута в VirtualBox или VMware из образа Ubuntu Server и загружается |
| 2 | Создан пользователь с именем по фамилии студента |
| 3 | SSH по ключу для этого пользователя работает, парольная аутентификация отключена |
| 4 | Описаны основные параметры ВМ (CPU, RAM, диски, сеть) |
| 5 | Раздел создан на /dev/sda, ext4, постоянное монтирование в `/disk`, доступ для созданного пользователя |

Лабораторная работа засчитывается при выполнении всех заданий.
